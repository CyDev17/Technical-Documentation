HTML Basics

HTML stands for HyperText Markup Language.

It is used to structure content on the web.

<h1>Hello</h1>

Basic tags include headings, paragraphs, links, etc.

  • Heading Tags
  • Paragraph Tags
CSS Styling

CSS stands for Cascading Style Sheets.

It is used to control the layout and appearance.

body { background-color: #fff; }

You can apply styles via inline, internal, or external CSS.

  • Selectors
  • Properties
JavaScript Syntax

JavaScript adds interactivity to web pages.

It is a scripting language that runs in the browser.

console.log("Hello World");

Variables can be declared using var, let, or const.

  • Functions
  • Events
DOM Manipulation

DOM stands for Document Object Model.

It represents the structure of an HTML document.

document.getElementById("demo").innerHTML = "Hi!";

You can add, delete, or change HTML elements using JS.

  • Selectors
  • Events
Responsive Design

Responsive design ensures your site looks good on all devices.

Media queries allow for conditional styling.

@media (max-width: 600px) { body { font-size: 14px; } }

Use relative units like %, em, and rem for better scaling.

  • Media Queries
  • Flexbox/Grid
